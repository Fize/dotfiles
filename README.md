# dotfiles

dotfile and configuration for my development environment.

## Requirements

- git

### Features

- zsh (with Oh My Zsh)
- neovim
- tmux

### Supported Systems

- **Linux**: Ubuntu, Debian, Fedora
- **macOS**: Darwin

### Linux And MacOS

- ghostty

### Windows

- wsl2
- Windows Terminal

### Install

```
 ./install.sh
```

### Neovim Plugins

use `:LazyExtra` to install the plugins

- copilot
- gitui
- aerial
- neo-tree
- telescope
- prettier
- leap
- fzf
- toggleterm
